# Setup

> ### 1. Clone the repo:
```bash
git clone https://github.com/Zephryon-zv/C-Programs.git
```

> ### 2. After making changes add all the changes:
```bash
git add .
```

> ### 3. Commit the changes:
```bash
git commit -m "Commit Message"
```

> ### 4. Push the changes:
```bash
git push origin main
```
